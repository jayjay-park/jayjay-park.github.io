Project Description

In (Jeonju ICT Innovation AI Kaggle Challenge), I created a model that classifies poker hand types.
(You can access to my Kaggle Notebook through this Link )

Training Data

Creating the model was challenging because the training data was oversampled, which means that the dataset is insufficient to represent its true domain. In this case, the number of training dataset of each categories decreased in descending order.

10 Poker hand categories:
  1. Nothing in hand; not a recognized poker hand, 12493 instances (49.95202% / 50.117739%)
  2. One pair; one pair of equal ranks within five cards, 10599 instances, (42.37905% / 42.256903%)
  3. Tow pairs; two pairs of equal ranks within five cards, 1206 instances, (4.82207% / 4.753902%)
  4. Three of a kind; three equal ranks within five cards, 513 instances, (2.05118% / 2.112845%)
  5. Straight; five cards, sequentially ranked with no gaps, 93 instances, (0.37185% / 0.392465%)
  6. Flush; five cards with the same suit, 54 instances, (0.21591% / 0.19654%)
  7. Full house; pair + different rank three of a kind, 36 instances, (0.14394% / 0.144058%)
  8. Four of a kind; four equal ranks within five cards, 6 instances, (0.02399% / 0.02401%)
  9. Straight flush; straight + flush, 5 instances, (0.01999% / 0.001385%)
  10. Royal flush; ace, king, queen, jack, ten + flush, 5 instances, (0.01999% / 0.000154%)

Considering such feature of the training dataset, followings are the approach I used:
  • Framework used: Keras
  • Sequential Model: used softmax for activation function as it is multi-classification model
  • Model Compile: used adam(Adaptive Moment Estimation) optimizer and categorical_crossentropy loss function
  • Model Fit: intentionally decreased the batch size and made shuffle equal to True to order to randomize

#Model Sequential
import keras
from keras.models import Sequential
from keras.layers.core import Dense, Dropout

model = Sequential()
model.add(Dense(128, activation='relu', input_dim=10))
model.add(Dense(64, activation='relu'))
model.add(Dense(32, activation='relu'))
model.add(Dense(16, activation='relu'))
model.add(Dropout(0.2))
model.add(Dense(10, activation='softmax'))


#Model Compile
from keras.optimizers import Adam
model.compile(loss='categorical_crossentropy', optimizer = Adam(lr=0.001), metrics=['accuracy'])

#Model Fit
history = model.fit(X_train, Y_train, epochs=800, batch_size= 32, validation_split= 0.2, shuffle=True)
